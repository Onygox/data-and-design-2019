Data: Star Classification¶

Data from https://www.kaggle.com/deepu1109/star-dataset

Our target is the star's type.

There are 6 classes:

Brown Dwarf -> Star Type = 0

Red Dwarf -> Star Type = 1

White Dwarf-> Star Type = 2

Main Sequence -> Star Type = 3

Supergiant -> Star Type = 4

Hypergiant -> Star Type = 5

Results:¶

Using K Nearest Neighbor & train-test-split, with data Radius and Absolute magnitude:

  • Training score is 96.6%
  • Test score is 88.3%

Using Logistic Regression & train-test-split, with data Radius and Absolute magnitude:

  • Training score is 90.5%
  • Test score is 85%

Using K Nearest Neighbor & train-test-split, with data Radius, Absolute magnitude, and Luminosity:

  • Training score is 80.5%
  • Test score is 73.3%

Using Logistic Regression & train-test-split, with data Radius, Absolute magnitude, and Luminosity:

  • Training score is 56.6%
  • Test score is 55%

Getting dummies will prob not help with this data but I tried it anyway

Using K Nearest Neighbor & train-test-split, with data from Spectral Class:

  • Training score is 36.1%
  • Test score is 35%
