College Graduation Rates¶

Description¶

Data includes college data including teacher/pupil ratio and graduation rates

Target = graduation rates

Exploration and descriptive analysis¶

  • Are there missing values?
  • Any feature encoding?
  • Any feature engineering?

Feature selection¶

  • correlation
  • sklearn
  • feature selection

Model 1¶

  • LinearRegression

Model 2¶

  • LinearRegression with PolynomialFeatures

Model 3¶

  • Grid Search CV with Pipeline

Model 4¶

  • Get dummies on the "Is_Private" data and extrapolate

Summary¶

  • Using a LinearRegression and "Out-of-state tuition", Root Mean Squared Error is 30.557800507931997
  • Using a LinearRegression and "Top10perc" (Pct. new students from top 10% of H.S. class), "Top25perc" (Pct. new students from top 25% of H.S. class) and "perc.alumni" (Pct. alumni who donate), Root Mean Squared Error is 13.957955215160267
  • Using Train-Test-Split and Grid Search CV with 'Out-of-state Tuition' & 'Pct. new students from top 10% of H.S. class', RMSE is 11.651145402202511
  • Using a Power Transformer with 'Out-of-state Tuition' & 'Pct. new students from top 10% of H.S. class', RMSE is 13.556374491316712
  • Using a LinearRegression with the 'Yes_Private' data, RMSE is 16.167623244714516
  • Same result using a GridSearchCV with the same data
